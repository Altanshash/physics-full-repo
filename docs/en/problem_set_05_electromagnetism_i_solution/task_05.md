# Problem 5 – Capacitor: Energy and Force

## Given

We have a parallel plate capacitor:

- $S = 0.02 \ \text{m}^2$
- $d = 5 \ \text{mm} = 5 \times 10^{-3} \ \text{m}$
- $U = 500 \ \text{V}$

Vacuum permittivity:

```math
\varepsilon_0 = 8.85 \times 10^{-12} \ \text{F/m}
```

Find:

1. Capacitance
2. Energy stored in the capacitor
3. Electric field intensity between the plates
4. Field energy density
5. Force of attraction between the plates

---

## Solution

## 1. Capacitance

For a parallel plate capacitor:

```math
C=\varepsilon_0\frac{S}{d}
```

Substitute the values:

```math
C=
(8.85\times10^{-12})
\frac{0.02}{5\times10^{-3}}
```

```math
C=3.54\times10^{-11}\ \text{F}
```

---

## 2. Energy stored in the capacitor

Energy stored in a capacitor:

```math
W=\frac{1}{2}CU^2
```

```math
W=
\frac{1}{2}(3.54\times10^{-11})(500)^2
```

```math
W=4.43\times10^{-6}\ \text{J}
```

---

## 3. Electric field intensity

Electric field between parallel plates:

```math
E=\frac{U}{d}
```

```math
E=
\frac{500}{5\times10^{-3}}
```

```math
E=1.0\times10^5\ \text{V/m}
```

---

## 4. Field energy density

Energy density of the electric field:

```math
w=\frac{1}{2}\varepsilon_0 E^2
```

```math
w=
\frac{1}{2}(8.85\times10^{-12})(1.0\times10^5)^2
```

```math
w=4.43\times10^{-2}\ \text{J/m}^3
```

---

## 5. Force of attraction between the plates

Force between capacitor plates:

```math
F=\frac{1}{2}\varepsilon_0 E^2 S
```

```math
F=
\frac{1}{2}(8.85\times10^{-12})(1.0\times10^5)^2(0.02)
```

```math
F=8.85\times10^{-4}\ \text{N}
```

---

## Final Answer

```math
C=3.54\times10^{-11}\ \text{F}
```

```math
W=4.43\times10^{-6}\ \text{J}
```

```math
E=1.0\times10^5\ \text{V/m}
```

```math
w=4.43\times10^{-2}\ \text{J/m}^3
```

```math
F=8.85\times10^{-4}\ \text{N}
```

---

## Conclusion

The capacitance of the parallel plate capacitor depends on the plate area and the distance between the plates.  
The electric field is uniform between the plates.  
The capacitor stores electric energy, and the plates attract each other with a small force.
